# pl-docker

##　https://github.com/lllyasviel/PaintingLight　をdockerで動かしたいときのメモ


###  とりあえずクローン
適当にディレクトリを作ってそこにクローンしてください。
'''
git clone https://github.com/lllyasviel/PaintingLight.git
'''
### docker hubからイメージを取得
'''
sudo docker login
'''
でログインして、

'''
sudo docker pull mojako/paintinglight'
'''
でイメージを持ってきてください。
